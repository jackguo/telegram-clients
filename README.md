# Ngram
A forked CLI telegram client. Refactored and revamped.

**Dependencies**

    tdlib 1.8.0

Building on Unix/Linux
----------
**Source**

    git clone --recursive https://github.com/alimovlex/ngram

**Build**

    cd ngram
    mkdir build
    cd build
    cmake ..
    make

**Run**

    ./ngram
